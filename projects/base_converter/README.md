# Base Converter（进制转换器）

## 一、项目目标

编写一个简单的命令行工具，实现不同进制之间的转换，当前阶段重点实现：

* 二进制 → 十进制
* 十进制 → 二进制

该项目用于巩固以下基础能力：

* 变量与数据类型（字符串 / 整数）
* 输入输出（input / print）
* 基本运算（加法、乘法、幂运算、整除、取余）
* 字符串处理
* 循环结构（后续实现）
* 条件判断（后续实现）

---

## 二、功能设计（当前阶段：仅设计，不实现）

### 功能 1：二进制 → 十进制

输入：

```
1011
```

输出：

```
11
```

要求：

* 输入为字符串形式的二进制数
* 逐位处理，每一位参与计算
* 按如下规则转换：

数学本质：

> 每一位 × 对应的 2 的幂，然后求和

---

### 功能 2：十进制 → 二进制

输入：

```
11
```

输出：

```
1011
```

要求：

* 输入为整数
* 使用“除 2 取余法”
* 结果需要倒序排列
